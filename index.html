Why it exists

A PDF editor for the small jobs people actually need.

Most PDFs do not need a giant document suite. They need a clear editor that helps you fix text where possible, add content, sign, review, manage pages, and save the result without turning a simple task into a workflow.

RewirePDF is built as an open project around practical PDF behavior: the interface stays approachable, while the editor still exposes the document details that matter when a PDF is complex.

Tools

The basic PDF operations, made comfortable.

RewirePDF focuses on operations that belong in a friendly PDF editor: small edits, annotations, signatures, page work, and export.

Edit existing text

Change source text in compatible PDFs while keeping the document editable and downloadable.

Add content

Place text, images, stamps, and corrections with clear formatting controls.

Sign documents

Type, draw, or import a signature and position it where the PDF needs it.

Review and annotate

Highlight, draw, comment, and mark up files during everyday review work.

Manage pages

Rotate, reorder, organize, and export the PDF after the document is ready.

Redact when supported

Use real redaction paths when the PDF structure allows the content to be removed.

Read certificates

Inspect certificate information so signed PDFs are not treated as a mystery box.

Handle permissions

Surface document permissions and restrictions where the PDF defines them.

PDF support

Made for real PDFs, not only perfect demo files.

PDF is an old, broad, sometimes strange format. RewirePDF aims to support the documents people actually receive: standard PDFs, files produced by office tools, Adobe-flavored documents, and XFA forms.

  • Adobe and XFA PDFs Open and work with more of the documents that arrive from real workflows.
  • Certificate reading Expose certificate details when a PDF carries signing information.
  • Permission awareness Show what the document says about printing, copying, editing, and access.
  • Clear edit limits When a PDF structure blocks an operation, the editor should say so clearly.

Platforms

Choose how and where to use it.

RewirePDF is not meant to trap you in one surface. The project is available on the web, works from mobile browsers, can be installed as an app-like PWA, and has native app work in progress.

Web editor

Open the editor from a browser and start with a PDF whenever you need a quick edit.

Mobile supported

Use RewirePDF from supported mobile browsers for lightweight PDF work away from a desktop.

Native app in development

Desktop-native packaging is part of the project direction, so the editor can meet different working habits.

Open project

Open source, built for people who want a better everyday PDF tool.

RewirePDF extends PDF.js with editing and saving features. The goal is a practical community project: readable code, visible development, and an editor that improves because people can inspect it, report issues, and shape the roadmap.
